# Structured

Personal blog on exploring unstructured data and related topics by Srikanth Pasumarthy.

## Running Locally

To run this Jekyll site locally on your machine:

1.  **Install Dependencies**:
    Ensure you have Ruby installed. Then run:
    ```bash
    bundle install
    ```

2.  **Start the Server**:
    Run the following command to start the local Jekyll server:
    ```bash
    bundle exec jekyll serve
    ```

3.  **View the Site**:
    Open your browser and navigate to:
    [http://127.0.0.1:4000](http://127.0.0.1:4000)
